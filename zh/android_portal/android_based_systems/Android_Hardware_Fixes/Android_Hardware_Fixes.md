# Android 硬件修复



## 修复 Android 硬件

### G1 硬件修复

#### 加速度传感器停止工作

如果加速度传感器停止工作，请尝试下面的方法：

-   尝试移除 "akmd\_set.txt" 文件来唤醒。方法是：
    -   移除 /data/misc/akmd\_set.txt 并重启手机 (或杀掉
        akmd 进程)
